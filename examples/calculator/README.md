# Calculator Plugin

Simple calculator plugin for Volt.

## Features

- Basic math operations (+, -, \*, /)
- Works with natural language
- Instant results as you type

## Usage

Type any math expression:

- `2 + 2` → 4
- `15 * 3` → 45
- `100 / 4` → 25
